基于yoman的webpack与react环境安装
安装yoman: npm install -g yo
进入磁盘：
generator: npm install -g generator-react-webpack
     检查: npm ls -g --depth=1 2>/dev/null | grep generator

远程github创建库：react   选用 MiT License
本地拉取并安装打包： yo react-webpack react

运行项目：npm start || npm run serve

css浏览器前缀处理：npm install autoprefixer-loader --save-dev

webpack -w 实时打包
webpack -p 代码压缩
webpack -d 解压缩
